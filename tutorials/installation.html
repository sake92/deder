Installation 🔗

MacOS and Linux (Homebrew) 🔗

brew tap sake92/tap
brew install deder

Windows 10+ 🔗

scoop bucket add sake92 https://github.com/sake92/scoop-bucket.git
scoop install deder

Manual installation 🔗

  1. Download deder-client.jar (or a native executable for your OS) from the early release
  2. Rename it to just deder:
mv deder-client.jar deder
  1. Make it executable:
chmod +x deder
  1. Put it in your PATH, e.g. by moving it to /usr/local/bin/:
sudo mv deder /usr/local/bin/
