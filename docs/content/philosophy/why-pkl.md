# Why Pkl?

Scala builds can become very complex.
It is not uncommon to have many combinations on multiple axis:
- scala version
- platform: JVM/JS/Native
- OS: linux/mac/win

This makes JSON/YAML/XML not so attractive.
You eventually have to invent new syntax for loops, conditionals etc.

Pkl makes this easier, it has types, docs, IDE integration etc.
